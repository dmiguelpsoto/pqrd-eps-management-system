# PQRD EPS Management System
Sistema web para la gestión de PQRD en EPS con integración de múltiples canales, análisis con IA y generación de reportes operativos.
## Funcionalidades
- Registro de PQRD desde múltiples canales
- Clasificación automática mediante IA
- Gestión de respuestas
- Alertas por vencimiento
- Reportes e indicadores operativos
## Tecnologías
Backend:
FastAPI
PostgreSQL

Frontend:
React

IA:
Procesamiento de lenguaje natural para clasificación de motivos

En desarrollo como proyecto de portafolio profesional.
